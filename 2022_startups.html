2022 Startups data visualization

This project used Tableau to gain insights of startups in 2022.

Photo by Clay Banks on Unsplash

Background

I did an interactive dashboard of 2022 Startups using the data that I downloaded from Kaggle.

1.AIMS grid


I used the AIMS grid to plan out my dashboard.


Purpose -> to see the insights of Startups in 2022


Stakeholders -> anyone who is interested to see Startups in 2022


End result -> an interactive dashboard where the user can see the Startups details of each country.


Success Criteria -> able to see which industries the Startups is in each country when the user selects and I able to see the valuation of each Startups in the country that I selected.

2.Interactive Dashboard

This interactive dashboard allows users to see the StartUps industries and each of the companies' valuations in the country chosen by the user. Click the link below to go to the dashboard.
