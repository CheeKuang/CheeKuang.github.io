Automated Gym Booking

This project used selanium for automated gym booking for gymboxx. Once the time reaches 8pm, the project will run automatically and booked the 8pm time slot for the user for the following day.

Photo by Sushil Ghimire on Unsplash

Background


This is the booking page for Gymboxx users to book their desired time slots and location of the gym.

1.Importing the libaries and Preparation


Firstly, i will import the libaries required for the automation and key in the details required for booking of gym.
*Mobile Number,Email and Gymboxx ID is not in the screenshot due to privacy issues*

2.Initializing Chrome Webdriver

3.Automated Booking

In the while loop, the if loop will check if the current time was greater than the booking time, which is 8 pm. if the current time is greater than 8 pm, the driver will find the element that has the outlet and then click it. The driver will then select the date and the timeslot. Lastly, it will proceed to key in the mobile number and the gymboxx id and click on the submit button.

4.Booking Success

Once booking is success, an otp will be send to the user to confirm the booking
